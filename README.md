###  Mini MERN App (`/mini-mern-app`)
A complete MERN stack application developed to manage employee records through a React frontend and a Node.js/Express backend connected to MongoDB.

#### Features
- Create new employee records
- View all employees
- Edit employee details
- Delete employee records

#### Concepts Practiced
- Building RESTful APIs with Express.js
- Designing MongoDB schemas using Mongoose
- Connecting React to backend APIs using Axios
- Navigation using React Router
- Handling CORS and environment variables during deployment

##  Key Learnings

### React Hook Form
Built forms with validation and error handling to write cleaner and more maintainable code.

### Mongoose and MongoDB
Created schemas and performed CRUD operations efficiently using Mongoose.

### Full-Stack Integration
Connected a React frontend with an Express backend to exchange data over HTTP.

### Deployment and Configuration
Configured environment variables (`process.env` and `import.meta.env`), resolved CORS issues, and deployed both frontend and backend successfully.
